## 데이터 종류 ==> 데이터 타입 (1개씩만 - 프리미티브 데이터 타입)
   수치 데이터 : 정수형 int 실수형 float
   문자 데이터 : 문자형 str
   논리 데이터 : boolean (True, False)

## 타입을 변경하는것 ==> 형변환/ 타입캐스팅/ 캐스팅

## 여러개의 데이터를 저장하는 데이터 타입 ==> 리스트, 튜플 (콜렉션 데이터 타입)
   - 리스트 : 원소 수정 가능 [   ]
   - 튜  플 : 원소 수정 불가 (10,) 10,

## 원소들이 순서가 있고 반복 가능한 데이터 타입 ==> 시퀀스 데이터 타입
   - 슬라이싱, 인덱싱 가능
   - 인덱스
   - 리스트 (수정O), 튜플, 문자열 (수정X)

## 반복 가능한 데이터 타입 ==> iterable
   - 시퀀스 데이터 타입 포항
   - 내부에 next() 함수를 사용해서 다음 원소, 다음 원소 ... 추출

## range() : 수의 범위를 만들어 내는 내장함수
   - range(시작숫자, 끝숫자+1, 간격)
