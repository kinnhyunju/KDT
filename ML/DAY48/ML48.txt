파라미터
- 모델 파라미터 
    : fit()학습 진행 후 설정되는 파라미터
    : 모델.파라미터명_ 
- 하이터 파라미터
    : 모델 인스턴스 생성 시 매개변수
    : 모델 인스턴스 생성할 때 개발가자 값을 설정
    : 모델 성능에 영향 미침
    : 예) 규제 강도 alpha, K 개수, 학습 횟수 max_iter, 규제강도 C(Cost / alpha의 역수)

- 튜닝
    : 모델의 성능을 높이는 작업, 즉 모델의 하이퍼파라미터의 값을 조절해서 성능 올리기
