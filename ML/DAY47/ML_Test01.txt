1. 기계학습 프로세스에 대해 간략하게 설명하세요
목표설정 - 데이터준비(수집, 추출, 가공) - 데이터 학습 - 모델 평가 - 활용(웹/앱)

2. 기계학습에 사용되는 데이터셋에 대해 설명하세요
학습용 데이터셋 : 규칙 / 패턴을 찾기위한 데이터
검증용 데이터셋 : 규칙 / 패턴을 찾는 과정 중 검사를 위한 데이터 (학습이 잘못 되었다면 중단하며, 학습이 잘되고 있다면 그대로 진행)
테스트용 데이터셋 : 규칙 / 패턴을 평가하기 위한 데이터 (학습이 잘 되었는지 검사함)

3. 지도학습의 특징을 설명하세요
명확한 정답이 주어진 데이터를 먼저 학습한 뒤 미지의 정답을 예측함 
데이터(피처)와 함께 타겟(라벨)을 같이 대입함

4. 데이터 전처리 중 처음 데이터 수집 후 데이터 정제하는 과정에 진행되는 작업을 설명하세요
결측치 처리, 이상치처리, 중복값처리, 노이즈처리(입력되지 않았는데 입력되었다고 여겨지는 값), 모순값처리

5. 모델 개념을 설명해 주세요
데이터가 가지는 일정학 규칙이나 패턴을 수식으로 만든 것

6. 모델의 결과 값이 수치를 예측해주는 기계학습은 무엇인가요
회귀

7. 모델 파라미터는 무엇인가요
파라미터란 모델이 데이터로부터 학습하는 값들을 말함 학습 데이터를 통해 업데이트 되고 모델의 예측을 결정 (선형회귀 모델 파라미터 - 가중치, 절편 등)

8. KNN 알고리즘에 대해 설명해 주세요
하나의 점을 설정해 모든 원소와 거리를 측정함 

9. 하이퍼 파라미터는 무엇인가요?
최적의 학습을 위해 직접 입력하는 파라미터들을 말함 

10. Regression 의 성능평가 지표들을 설명하세요
MSE : 평균제곱오차함수
RMSE : MSE의 제곱근
MAE : mean absolute error
R2score : 회귀계수

11. LinearRegression 알고리즘에 대해 설명해 주세요
데이터 분포가 선형인 경우 해당 데이터를 가장 만족하는 선형 관계식을 찾아서 새로운 데이터에 대한 타겟값을 예측하는 알고리즘

12. Classification 의 성능평가 지표들을 설명하세요
Accuracy score(정확도) : 전체 문제 중에 답을 맞춘 비율
ROC-AUC score : 다양한 임계값에서 모델의 분류 성능에 대한 측정 그래프

13. 피쳐공학에 대해 설명하세요
머신러닝 모델을 위한 데이터 컬럼을 생성하거나 선택하는 작업으로 전처리 단계에서 진행함
Feature가 많은 경우 발생하는 문제점을 해결하고 성능 개선을 하기 위해 필요함
차원축소 또는 피처 생성

14. LogisticRegression 알고리즘에 대해 설명해 주세요
선형회귀방식을 분류에 적용한 알고리즘으로 시그모이드 함수 최적선을 찾고 그 반환값을 확률로 간주해 확률에 따라 분류를 적용함
=> 선형회귀 방식을 기반으로 하되 시그모이드 함수를 이용해 분류를 수행하는 회귀


15. 과대적합 과소적합에 대해 설명해 주세요
과대적합 : 훈련 데이터셋에만 최적화된 모델로 새로운 데이터에 대한 오차가 매우 크다 (학습을 너무 많이함)
과소적합 : 훈련 데이터셋의 규칙이나 패턴이 반영되지 않는 모델로 훈련데이터와 새로운 데이터에 대해 모두 오차가 매우 큼 (학습횟수가 부족하거나 데이터가 부족한 경우)
